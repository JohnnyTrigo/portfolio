## Case em destaque — Telecom Ops Analytics

Solução completa de Business Intelligence para operações de telecomunicações, unindo engenharia de dados, modelagem analítica e visualização executiva.

**Desafio:** A gestão operava sem visibilidade em tempo real sobre KPIs críticos. Dados fragmentados em múltiplas fontes impossibilitavam decisões rápidas.

**Solução:**
- Pipeline de dados com SQL avançado (CTEs, Window Functions, índices otimizados)
- Modelagem Star Schema no Power BI com medidas DAX avançadas
- Dashboard com RLS (Row-Level Security) para múltiplos perfis de gestão
- Automação de processos críticos com VBA Expert

**Resultados:**
| Métrica | Antes | Depois |
|---|---|---|
| Tempo de atualização | 7 horas | 4 minutos |
| Erros manuais | Recorrentes | Eliminados |
| Aderência a SLAs | Variável | 100% |
| Visibilidade de KPIs | Zero | Tempo real |
